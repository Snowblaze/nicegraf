\doxysubsection{Detailed Description}
Describes how blending should be handled by the pipeline. If blending is disabled, the resulting color and alpha values are directly assigned the color and alpha values computed at the fragment stage.

When blending is enabled, the resulting color and alpha values are computed using the corresponding blend operations and factors (specified separately for color and alpha). Note that if the render target attachment from which the destination values are read uses an s\+RGB format, the destination color values are linearized prior to being used in a blend operation.

If the render target attachment uses an s\+RGB format, the resulting color value is converted to an s\+RGB representation prior to being finally written to the attachment.
